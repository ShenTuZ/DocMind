# DocMind - 智能文档助手

DocMind是一款基于Electron框架开发的智能文档助手应用，集成了先进的AI技术，旨在提供高效、智能的文档管理和交互体验。

## 项目结构

- `electron-app/`：主应用目录
- `vosk-model-cn-0.22/`：中文语音识别模型（已在.gitignore中忽略）
- `q1-fy25-earnings.pdf`：测试文档，用于功能测试（已在.gitignore中忽略）

## 快速开始

请查看 `electron-app/README.md` 文件了解详细的安装和使用说明。

## 技术特点

- **跨平台**：基于Electron框架，支持Windows、macOS和Linux
- **智能交互**：集成AI模型，支持自然语言处理和多模态输入
- **文档处理**：支持多种文档格式的分析和处理
- **语音识别**：集成Vosk语音识别引擎，支持语音输入
- **知识库管理**：基于RAG技术构建和管理知识库
